# North Star

A Raspberry Pi should feel like a Bluetooth speaker.

You plug it in.

It appears.

You tap it.

It works.

You never wonder what network it is on.

You never search for an IP address.

You never configure a tunnel.

You never think about SSH.

You simply use your Raspberry Pi.

---

Every line of code, every protocol decision, every animation, every document, and every feature should move BuzzPi closer to that experience.

If a decision does not serve this vision, it does not belong in the platform.

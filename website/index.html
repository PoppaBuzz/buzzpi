Features

Everything you need to manage your Pi.

Terminal. Remote desktop. File manager. Docker. GPIO. Camera. Monitoring. AI assistant. All from one platform.

Remote Desktop

Full graphical screen streaming. See and control your Pi’s desktop from anywhere.

One-Tap Terminal

Full PTY terminal with ANSI colors, multiple tabs, search. No SSH setup needed.

File Manager

Browse, upload, download, rename, preview files on your Pi. Drag and drop.

Docker Management

Containers, images, compose stacks, logs, resource monitoring at your fingertips.

GPIO Control

Interactive pin controls. Toggle pins, read sensors, visualize circuits in real time.

Camera Streaming

Live stream, snapshots, recording. View your Pi’s camera from your phone.

System Monitoring

CPU, memory, temperature, storage, network, uptime — everything at a glance.

AI Assistant

Diagnose issues, read logs, restart services, answer questions about your Pi.

How It Works

Three steps. Zero networking knowledge.

From unboxing to connected in under 60 seconds. No terminal, no config files, no frustration.

1

Plug In

Install BuzzPi Agent on your Raspberry Pi with a single command. That’s it.

2

Discover

Your Pi appears automatically on your phone, desktop, and CLI. No IP needed.

3

Tap

Connect instantly. Terminal, remote desktop, file manager — all from one interface.

Architecture

Open platform. Open protocol.

BuzzPi is built on BPP, the BuzzPi Protocol — a JSON-based protocol over WebSocket that any device or client can implement.

BuzzPi Agent

Go daemon running on each Pi. Identity, discovery, terminal, plugins.

BuzzPi Android

Native Android app. Jetpack Compose, Material 3. Your Pi in your pocket.

BuzzPi CLI

Cross-platform command-line client for scripting and power users.

BuzzPi Cloud

Relay and registry services. Secure off-LAN access without port forwarding.

BuzzPi SDK

Plugin and integration SDK. Extend BuzzPi with custom capabilities.

BPP Protocol

The BuzzPi Protocol. JSON over WebSocket. The contract that makes it all work.

Any device that implements BPP can participate in the ecosystem.

# Book 8: Lexicon

**What do we call things?**

The BuzzPi Lexicon defines the project's controlled vocabulary. Every term, every verb, every adjective used in the platform has exactly one canonical definition. Developers, designers, writers, and contributors use the same words to mean the same things.

Consistency is not cosmetic. It is structural. When every client, every document, and every error message uses the same language, the platform feels inevitable rather than assembled.
